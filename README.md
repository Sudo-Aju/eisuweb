<h1>EISU - Essential Wear</h1>

<p>A minimalist, single-page e-commerce landing page built with pure HTML and CSS for a modern apparel brand.</p>

<h2>Overview</h2>
<p>This project is a static frontend template designed to showcase a premium clothing collection. It features a sticky navigation bar, a full-screen hero section with a background image, and a responsive product grid displaying items and prices.</p>

<h2>Features</h2>
<ul>
    <li><strong>Responsive Design:</strong> Utilizes CSS Grid to automatically adjust the product layout for mobile, tablet, and desktop screens.</li>
    <li><strong>Zero Dependencies:</strong> Built entirely with vanilla HTML and embedded CSS. No frameworks, external stylesheets, or JavaScript libraries are required.</li>
    <li><strong>Modern Aesthetic:</strong> Uses a stark black and white color palette, bold borders, and uppercase typography to achieve an edgy, high-end streetwear look.</li>
    <li><strong>Sticky Navigation:</strong> The top header remains visible and accessible at the top of the screen as the user scrolls down the page utilizing <code>position: sticky</code>.</li>
    <li><strong>Semantic Structure:</strong> Employs standard HTML5 tags like <code>&lt;nav&gt;</code>, <code>&lt;header&gt;</code>, and <code>&lt;footer&gt;</code> for improved code readability and SEO.</li>
    <li><strong>Hero Image Overlay:</strong> Applies a dark linear gradient over the background image to ensure the primary hero text remains legible regardless of the image used.</li>
    <li><strong>Interactive Hover States:</strong> Includes a smooth scale transformation and color inversion effect on the primary call-to-action button to improve user feedback.</li>
    <li><strong>Universal Box Sizing:</strong> Applies <code>box-sizing: border-box</code> globally to guarantee consistent element padding and dimensions across the entire page layout.</li>
    <li><strong>Text Shadowing:</strong> Enhances the main hero text visibility against complex backgrounds with a subtle drop shadow effect.</li>
    <li><strong>Optimized Image Handling:</strong> Contains the navigation logo within a fixed-height container using <code>overflow: hidden</code> to prevent layout shifts.</li>
</ul>

<h2>How to Use</h2>
<p>Because this project contains no server-side code or complex build steps, viewing it is straightforward:</p>
<ol>
    <li>Download or copy the HTML code into a single file named <code>index.html</code>.</li>
    <li>Double-click the <code>index.html</code> file to open it in any modern web browser (such as Chrome, Firefox, Safari, or Edge).</li>
    <li>ORr you can just visit the site visible in the github description of the repo on the right!</li>
</ol>

<h2>Technologies Used</h2>
<ul>
    <li>HTML5</li>
    <li>CSS3</li>
</ul>

<h2>Customization</h2>
<p>To customize the template for your own brand, you can easily modify the following elements within the code:</p>
<ul>
    <li>Update the image links in the <code>&lt;nav&gt;</code> and <code>&lt;header&gt;</code> sections to use your own logo and hero background.</li>
    <li>Modify the text inside the product grid (<code>&lt;div class="p"&gt;</code>) to display your own product names and prices.</li>
    <li>Adjust the hex color codes in the CSS to match your brand's specific color scheme.</li>
</ul>
